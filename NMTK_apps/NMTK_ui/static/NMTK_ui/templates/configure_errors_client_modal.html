Configuration Errors

One or more of your configuration fields contains errors, which most likely fall into the following criteria:

  • One or more required fields is empty
  • One or more fields contains an invalid value

It is possible that you may need to expand one or more configuration sections to locate those fields that have errors.

Note: The "submit" button will change from orange to green when all fields are valid, and error messages now appear below any fields that contain errors.
