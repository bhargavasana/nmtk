NMTK Introduction

The Non-Motorized Transportation Toolkit (NMTK) is a flexible tool that allows you to use a wide range of models for transportation planning via a flexible interface.

The Non-Motorized Travel Analysis Toolkit provides access to a number of analysis tools that you can apply to your own data, initially including models for direct demand estimation both on roadway links and at intersections.

These are the steps for using the toolkit:

  • Begin by exploring the tools (below):
    • Read the tool documentation to learn about what the tool does and how it was developed.
    • Familiarize yourself with the data required for a tool by loading and examining the tool's sample data and sample job.
  • Prepare and upload the necessary data for your selected tool.
  • Start a new job, filling in the necessary job parameters.
  • Submit the job to run it (your data will be uploaded to the tool, and you will be notified when the results are ready.)
  • View the results, which can be accessed in one of the following ways:
    • In your browser (for spatial data, you can see it on a map!)
    • As a Microsoft Excel Spreadsheet (XLS) file.
    • As a Comma-Separated Values (CSV) file.
    • As a GeoJSON file (the "native" format that NMTK uses for data interchange)
