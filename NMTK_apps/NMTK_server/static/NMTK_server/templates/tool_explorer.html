Tool Explorer Introduction

The tool explorer allows you to browse tools that are provided by this server, review their documentation and/or supporting research papers, and download sample data sets.

In order to use a tool described here, you will need to do the following:

  • Prepare and upload the necessary data for your selected tool.
  • Start a new job, filling in the necessary job parameters.
  • Run the job (your data will be uploaded to the tool, and you will be notified when the results are ready.)
  • View the results, which can be accessed in one of the following ways:
    • As a Microsoft Excel Spreadsheet (XLS) file.
    • As a Comma-Separated Values (CSV) file.
    • In your browser (for spatial data, you can see it on a map!)
    • As a GeoJSON file (the "native" format that NMTK uses for data interchange)

Below is a list of Tools currently provided by this NMTK Server, you can read about the tools and download sample data files. When you are ready to try running your own job, click on the My Jobs tab to create and run your own NMTK job.
