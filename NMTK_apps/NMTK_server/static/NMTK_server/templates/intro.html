NMTK Introduction

The Non-Motorized Travel Analysis Toolkit provides access to a number of analysis tools that you can apply to your own data, include models for direct demand estimation both on roadway links and at intersections, and for assessing the quality of service on bicycle and pedestrian facilities using methods from the 2010 Highway Capacity Manual.

These are the steps for using the toolkit:

  • Begin by exploring the tools (below):
    • Read the tool documentation to learn about the models available and (optional) model tuning variables.
    • Familiarize yourself with the data required for a tool by downloading a sample.
  • Prepare and upload the necessary data for your selected tool.
  • Start a new job, filling in the necessary job parameters.
  • Run the job (your data will be uploaded to the tool, and you will be notified when the results are ready.)
  • View the results, which can be accessed in one of the following ways:
    • As a Microsoft Excel Spreadsheet (XLS) file.
    • As a Comma-Separated Values (CSV) file.
    • In your browser (for spatial data, you can see it on a map!)
    • As a GeoJSON file (the "native" format that NMTK uses for data interchange)

The Non-Motorized Transportation Toolkit (NMTK) is a flexible tool that allows you to leverage a wide range of models for transportation planning via a flexible interface.
